Projetos:

JBS Work Flow

JBS Work Flow é uma ferramenta desenvolvida por alunos do Germinare Tech com o objetivo de facilitar a organização e o acompanhamento das rotinas escolares. A aplicação oferece recursos como notificações automáticas, automação de tarefas, integração com inteligência artificial e visualização de informações por meio de widgets, promovendo mais produtividade no dia a dia acadêmico.

Git Script

git_start_script é um script Python simples criado para automatizar a inicialização de repositórios Git no Windows. Ideal para desenvolvedores que frequentemente criam novos repositórios e querem agilizar processos como init, add, commit, push e configuração do origin. Feito com foco em produtividade, o projeto conta com uma estrutura modular, suporte a execução via terminal ou atalho .bat e é ideal para quem trabalha com Git diariamente.

Project Setup 2.0

ProjectSetup é uma ferramenta automatizada para criação de projetos, desenvolvida para simplificar o início de novos trabalhos. A versão 2.0 do ProjectSetup traz melhorias importantes em relação à versão anterior, permitindo a criação não apenas de projetos em Python, mas também de sites e projetos Django, com uma interface de usuário interativa.
